# Ecommerce Backend

Backend API for the Ecommerce Project built with Node.js and Express.

## Features

- REST API built with Express.js
- Products API
- Shopping Cart API
- Orders API
- Payment Summary API
- Delivery Options API
- Serves static images
- CORS enabled for frontend deployment

## Tech Stack

- Node.js
- Express.js
- JavaScript

## Frontend

The frontend is deployed separately on Vercel:

[View Website](https://yks-ecommerce-project.vercel.app)

## API

Example:

GET /api/products

GET /api/cart-items

POST /api/cart-items

PUT /api/cart-items/:id

DELETE /api/cart-items/:id

## Run Locally

```bash
npm install
npm start
```

or

```bash
npm run dev
```
